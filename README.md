# Comparador Segunda Vuelta Presidencial Colombia 2026: Cepeda vs. De la Espriella

Plataforma interactiva y objetiva diseñada para comparar las hojas de vida, propuestas, controversias y mitos de los candidatos **Iván Cepeda** (Pacto Histórico) y **Abelardo de la Espriella** (Defensores de la Patria) con miras a la segunda vuelta presidencial del **21 de junio de 2026**.

El propósito del proyecto es brindar información verificada, estructurada y de fácil acceso, con citas directas a medios y fuentes primarias, permitiendo a los ciudadanos votar de manera informada.

---

## 🌟 Características de la Aplicación

- **Fácil Navegación:** Interfaz intuitiva y moderna (diseño premium y responsivo) con barra de navegación interactiva.
- **Hoja de Vida Dinámica:** Comparación de datos académicos, cargos públicos y trayectoria profesional.
- **Propuestas Temáticas:** Análisis detallado de propuestas divididas por sectores clave (Economía, Justicia, Paz, entre otros).
- **Mitos y Verdades:** Desmentido de noticias falsas y aclaración de rumores con fuentes verificadas.
- **Controversias y Contradicciones:** Sección dedicada a los cuestionamientos éticos y políticos de ambos candidatos, con soporte documental.
- **Visualización de Apoyos y Alianzas:** Mapeo de figuras políticas y partidos que respaldan cada candidatura.
- **Optimización Multimedia:** Carga optimizada de imágenes dinámicas utilizando almacenamiento en la nube (Vercel Blob).

---

## 🛠️ Tecnologías Utilizadas

- **Core:** [Next.js](https://nextjs.org/) (versión 16 con App Router) y [TypeScript](https://www.typescriptlang.org/).
- **Estilos y Animaciones:** [Tailwind CSS v4](https://tailwindcss.com/) y [Framer Motion](https://www.framer.com/motion/) para micro-interacciones fluidas.
- **Componentes e Iconografía:** [Lucide React](https://lucide.dev/) para iconos de interfaz limpia.
- **Gráficos y Datos:** [Recharts](https://recharts.org/) para visualización interactiva.

## 🚀 Inicio Rápido en Entorno Local

Sigue estos pasos para ejecutar el proyecto de forma local:

### 1. Clonar el repositorio
```bash
git clone https://github.com/AmunM9/cepeda-vs-abelardo.git
cd cepeda-vs-abelardo
```

### 2. Instalar dependencias
```bash
npm install
```

### 3. Ejecutar el servidor de desarrollo
```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) (o el puerto configurado en consola) en tu navegador para ver la aplicación corriendo.

### 4. Generar la compilación para producción
```bash
npm run build
```
